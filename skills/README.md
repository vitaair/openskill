# Skills

## codex/dev-perception-compiler

The development perception compiler is a Codex-oriented skill that turns unstructured development signals into actionable agent work:

- goals and tradeoffs into natural language
- task organization into Markdown
- engineering context into paths, code, diffs, and tests
- machine interfaces into JSON/YAML schemas
- visual behavior into screenshots or browser verification
- long logs into failure points, evidence, and excluded noise
