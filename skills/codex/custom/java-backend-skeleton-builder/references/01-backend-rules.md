# Backend Rules

## Data Access

Default:

```text
MyBatis
```

Supported:

```text
MyBatis-Plus
JPA
```

## Database

Default:

```text
MySQL 8
```

Supported:

```text
PostgreSQL
OceanBase
```

## JSON

Use Jackson through Spring Boot.

Do not default to fastjson.

## Utilities

Use JDK/Spring first.

Do not default to Hutool.
