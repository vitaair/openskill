# Rhetorica ad Herennium

本地文件：`rhetorica-ad-herennium-english.pdf`

中文核心章节译稿：`rhetorica-ad-herennium-book3-memory-zh.md`

来源：https://lyceum.institute/wp-content/uploads/2026/02/Anonymous-RhetoricaAdHerennium_Digital.pdf

来源说明：Lyceum Institute 页面说明该版本是 Harry Caplan 1954 英译本的 public domain republication。原作《Rhetorica ad Herennium》是古典修辞学文本，其中 Book III 包含早期、系统的人工记忆和 method of loci 说明。

使用方式：

- 本书作为记忆宫殿 SKILL 的“原典层”参考。
- 后续优化记忆宫殿核心方法时，优先参考 Book III 中关于 `loci`、`images`、顺序、显著性和复习巡行的原则。
- 不把整本书内容复制进主规范；主规范只吸收可执行原则。
- 中文使用时优先读 `rhetorica-ad-herennium-book3-memory-zh.md`，该文件翻译 Book III 中关于记忆术的核心章节。

重点章节：

- Book III, Chapter XVI-XXIV：natural memory、artificial memory、loci、images、order、striking images、training artificial memory。
