翻译范围：Book III, Chapter XVI-XXIV，即关于 memory、artificial memory、loci、images 和 training artificial memory 的核心章节。

译稿性质：用于 AI/Agent 记忆宫殿 SKILL 的中文资料层和方法吸收。此版优先追求可读性和方法可迁移性，不作为学术校勘定本。

## 第十六章 论记忆

现在，我要转向“记忆”。它是发明所得思想的宝库，也是修辞各部分的守护者。

至于记忆是否具有某种可由技艺造成的性质，还是完全来自天性，我们可以在另一个更合适的时机讨论。眼下，我先把这件事视为已经成立：在记忆这件事上，技艺和方法具有很大作用；并据此来处理这个主题。就我而言，我确信存在一种记忆之术，至于我为什么这样相信，我将在别处说明。现在，我先说明记忆究竟是什么。

记忆有两种：一种是自然的，另一种是由技艺造成的。自然记忆是嵌在我们心灵中的记忆，它与思想同时产生。人工记忆则是通过某种训练和纪律体系而得到强化的记忆。

在其他事情中，天赋的卓越常常可以与后天学习相匹敌，而技艺又会反过来加强并发展自然的优势；在记忆上也是如此。一个人如果天生记忆力非常出色，他的自然记忆常常近似于人工记忆；而人工记忆也会通过训练的方法保存并发展自然的优势。

因此，自然记忆必须通过训练来加强，使它变得出众；另一方面，由训练提供的记忆也需要自然能力。就像其他技艺一样，学问依靠天赋而努力，天性也依靠技艺规则而发展。这里给出的训练，对于那些天生记忆力好的人也会有用，你很快就会明白这一点。即便这些人凭借天赋并不需要我们的帮助，我们仍有理由去帮助那些天赋较弱的人。现在，我要讨论人工记忆。

人工记忆包含“背景”和“图像”。所谓背景，是指那些天然或人为划分出来的小型场景，它们完整而醒目，使我们能凭自然记忆轻易把握并包住它们。例如一所房子、两柱之间的空间、一个凹室、一座拱门，或类似的东西。

所谓图像，仿佛是我们想记住之物的形体、标记或肖像。比如我们想回忆一匹马、一头狮子或一只鹰，就必须把它的图像放到一个确定的背景中。现在，我要说明我们应该发明什么样的背景，怎样发现图像，又怎样把图像安置进去。

## 第十七章 记忆地点

认识字母的人，能够把听到的话写下来，也能把自己写下的内容读出来。同样，学会记忆术的人，也能把听到的东西安放在背景中，并凭这些背景从记忆中说出来。

背景很像蜡板或纸草；图像像字母；图像的排列和布置像文字书写；讲述则像阅读。因此，如果我们想记住大量事项，就应当准备大量背景，好在其中安放大量图像。

我同样认为，背景必须排成一个序列。这样，我们才不会因为顺序混乱而无法跟随图像。无论从序列中的哪一个背景开始，无论向前还是向后，我们都能从其中取回已经交付给背景的内容，并把它说出来。

## 第十八章 地点的排序

举例来说，如果我们看到许多熟人按某种顺序站着，那么我们从队首、队尾或中间开始说出他们的名字，并不会造成什么差别。背景也是如此。

如果背景已经按顺序排列，那么结果就是：在图像的提醒下，我们可以从任意一个背景出发，沿任意方向，把先前交付给背景的内容口头复述出来。这就是为什么最好把背景排成序列。

我们必须特别认真地熟悉已经采用的背景，使它们能够长久地黏附在记忆中。因为图像像字母，如果不用就会被擦去；但背景像蜡板，应当保留下来。

为了避免在背景数量上出错，每第五个背景都应该做一个标记。例如，在第五个背景中放一只金手，在第十个背景中放一个名字叫 Decimus 的熟人。这样，在以后每一个第五背景中设置类似标记，就会很容易。

## 第十九章 地点的适宜性

再者，在荒僻之地取得背景，比在人多之处取得背景更有利。因为人群拥挤、来往穿行，会混乱并削弱图像的印痕；而清静会使图像轮廓保持鲜明。

此外，背景必须在形态和性质上彼此不同，这样经过区分之后，它们才会清楚可见。因为如果一个人采用了许多相似的柱间空间，它们彼此相像，就会使他混乱，最后不知道自己在每个背景中放了什么。

背景的大小和范围也应当适中。过大时，图像会变得模糊；过小时，又常常显得容不下图像的布置。

背景既不应太明亮，也不应太昏暗。太暗会让阴影遮蔽图像；太亮又会让光泽使图像晃眼。我认为，背景之间的距离也应适中，大约三十英尺。因为思想的内在眼睛和外在眼睛一样：观看对象太近或太远，力量都会减弱。

经验较多的人，很容易为自己准备足够多、足够合适的背景。即便有人觉得自己找不到足够好的背景，也仍然可以按需要构造出许多这样的背景。因为想象力能够包容任何区域，并能在其中随意塑造和建造某些背景的场所。

因此，如果现成的背景储备不能令我们满意，我们可以在想象中为自己创造一个区域，并取得一套非常有用的适宜背景分布。

关于背景，说到这里已经足够。现在让我转向图像的理论。

## 第二十章 与事项相称的图像

既然图像必须与对象相似，我们就应当从万物中选择可供使用的相似物。因此，相似物必然有两种：一种是事项的相似，一种是词语的相似。

事项的相似，是指我们使用能够概括呈现所处理事项的图像。词语的相似，则是指每一个单独的名词或称谓，都由一个图像保存其记录。

我们常常可以用一个标记、一个单一图像，来包含整个事项的记录。比如，控方说被告用毒药杀了一个人，又指控说犯罪动机是一笔遗产，并宣称有许多证人和同谋参与此事。如果为了便于辩护，我们想记住第一项指控，就应在第一个背景中构成整个事项的图像。

如果我们认识那个人，就把他想象成病卧在床。如果不认识他，就仍然选某个人作为病人，而且最好选一个地位最低下的人，使他能立刻浮现在心中。然后，我们把被告放在床边，让他右手拿杯子，左手拿书写板，第四根手指上戴着羊的睾丸。

这样，我们就能记录被毒害的人、遗产和证人。以同样方式，我们按顺序把其他指控事项依次放进背景中。每当我们想回忆某一点，只要妥善安排背景的样式，并仔细压印图像，就能轻易把我们想要的东西召回心中。

## 第二十一章 与词语相称的图像

当我们想用图像表现词语的相似时，我们要承担的任务更大，也更需要发挥机敏。方法应当如下。

例如有这样一句诗：

> Iam domum itionem reges Atridae arant.

意思是：“现在，他们的归家之行，阿特柔斯的子孙诸王，正在准备。”

如果我们想记住这句诗，就应在第一个背景中放入 Domitius：他举手向天，同时被 Marcii Reges 鞭打。这样可以表示 “Iam domum itionem reges”。在第二个背景中，放入 Aesopus 和 Cimber，让他们装扮成《伊菲革涅亚》中阿伽门农和墨涅拉俄斯的角色。这样可以表示 “Atridae parant”。

用这种方法，所有词语都能被表示出来。但这种图像安排只有在我们用标记来刺激自然记忆时才会成功。因此，我们应先把给定的诗句默念两三遍，然后再用图像表示这些词语。这样，技艺就会补充天性。

两者单独使用都不够强，不过必须注意，理论和技巧要可靠得多。若不是担心偏离原本计划，损害这份教导的清楚和简洁，我本来可以更详细地展示这一点。

在通常情况下，有些图像强烈、鲜明，适合唤醒回忆；另一些图像则软弱无力，几乎不能刺激记忆。因此，我们必须思考造成这些差异的原因。知道原因之后，我们才知道应避免哪些图像，追求哪些图像。

## 第二十二章 固定图像

自然本身会教我们该怎么做。在日常生活中，当我们看到琐碎、平常、普通的事物时，通常记不住它们，因为心灵没有被任何新奇或惊异之物激动。

但如果我们看到或听到某件特别卑劣、可耻、异常、巨大、难以置信或可笑的事情，我们很可能会记住很久。因此，眼前耳边的平常事我们常常忘记；童年发生的事件，我们却往往记得最牢。原因只能是：普通事物容易从记忆中滑走，而醒目和新奇之物会在心中停留更久。

日出、太阳运行、日落，谁都不会觉得惊奇，因为它们天天发生。但日食会令人惊奇，因为它很少发生；而且它比月食更令人惊奇，因为月食更常见。由此可见，自然不会被普通、平常的事件唤醒，却会被新的或醒目的事件打动。

那么，技艺就应当模仿自然，找到自然所欲求的东西，并按照自然的指引行动。因为在发明中，自然从来不是最后的，教育也从来不是最先的；事物的开端来自天赋，而终点则由训练抵达。

因此，我们应当设置那种最能长久黏附在记忆中的图像。要做到这一点，就应建立尽可能醒目的相似物；设置的图像不要数量太多，也不要模糊，而要让它们正在做某件事；赋予它们异常的美或特别的丑；例如给某些图像戴上王冠或披上紫袍，使相似物对我们更清楚；或者以某种方式使它们变形，比如引入一个沾满鲜血、污泥或红颜料的人，使形体更加醒目；也可以给图像安排某些滑稽效果，因为这同样会保证我们更容易记住它们。

当某些事物真实存在时，如果它们容易被我们记住，那么当它们只是虚构出来、但被仔细描绘时，我们也同样能够毫不困难地记住它们。

不过，有一件事必不可少：必须一遍又一遍地在心中快速巡行所有原初背景，以刷新其中的图像。

## 第二十三章 图像的个人差异

我知道，许多写过记忆术的希腊作者，会列出大量词语对应的图像，好让想学习这些图像的人可以不费力寻找，直接背熟备用。我出于几个理由不赞成这种方法。

第一，在不可计数的词语中，为一千个词收集图像是可笑的。因为在无限的词语储备中，我们有时需要记这个，有时又需要记那个；这区区一千个图像能有多大价值呢？

第二，为什么要剥夺任何人的主动性？为了让他不用自己寻找，就把所有已经找好的东西交给他，这并不合适。

再者，一个人会被某种相似物打动，另一个人却会被另一种相似物打动。事实上，当我们说某个形体像另一个形体时，常常不能得到所有人的同意，因为不同的人看事情不同。

图像也是如此。对我们而言轮廓鲜明的图像，对别人来说可能相当不醒目。因此，每个人在为自己准备图像时，都应适合自己的便利。

最后，教师的职责是在每一种情况下教人如何寻找，并为了更清楚起见，补充一两个同类例子，而不是把所有例子都列出来。比如，当我讨论怎样寻找引言时，我给出的是寻找的方法，而不是草拟一千种引言。我认为，对图像也应采用同样做法。

## 第二十四章 人工记忆的训练

你也许会认为，逐字记忆要么太难，要么用处太小，于是满足于记住事项，因为这更容易也更有用。为了避免这种情况，我必须说明，为什么我并不反对逐字记忆。

我相信，那些想毫不费力地完成容易事情的人，必须先在更困难的事情上接受训练。我纳入逐字记忆，并不是为了让我们死背诗句，而是把它作为一种练习，用来加强另一种具有实际用途的记忆，也就是事项记忆。这样，我们就能从艰难训练中轻松过渡到另一种记忆。

在每一种学科中，如果没有持续不断的练习，技艺理论几乎没有用处；尤其在记忆术中，如果不靠勤勉、专注、劳苦和细心来落实，理论几乎毫无价值。

你应当确保自己拥有尽可能多的背景，并让这些背景尽可能符合规则；在放置图像时，应当每天练习。繁忙的事务常常会使我们分心，不能从事其他学习；但没有任何事情能让我们完全离开这项活动。事实上，从来没有一个时刻我们不想把某些东西交付记忆，而当我们的注意力被特别重要的事务占据时，我们最想这样做。

既然敏捷的记忆是一件有用的东西，你就能清楚看出，我们必须付出多大努力，才能获得如此有用的能力。等你知道它的用途之后，就能理解这条建议的价值。

关于记忆，我不打算再劝勉你。否则，我看起来不是不相信你的热忱，就是对这个主题的讨论还不够充分。

接下来，我要讨论修辞的第五部分。你可以在心中复习前四个部分；尤其必要的是，用练习来巩固你对它们的掌握。

## 对 AI/Agent 记忆宫殿的直接启发

- 背景对应固定结构、文件目录、九区路线和 L0-L3 读取层级。
- 图像对应记忆卡片、标题、标签、触发场景、风险提示和行动提示。
- 背景要稳定，图像可更新；这对应“原文层/索引层分离”。
- 背景必须排序，才能从任意位置向前或向后恢复上下文。
- 每第五个背景做标记，可转化为章节锚点、里程碑卡片和周期审计点。
- 背景要适中、彼此区分、避免拥挤，可转化为 Markdown 文件拆分和标签治理规则。
- 图像要醒目、具体、正在行动，不能是模糊摘要。
- 图像应由使用者生成，不应完全依赖通用图像库；这对应不同项目的自定义记忆卡片。
- 人工记忆必须每天练习；对 Agent 来说，就是每次启动、交接、迁移和阶段结束时巡检记忆文件。
