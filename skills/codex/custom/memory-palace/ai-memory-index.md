本 SKILL 不只面向 Codex，也为 Cursor、Claude Code、Gemini、Cline、Windsurf 等工具提供轻量入口。

### 本目录入口文件

- `SKILL.md`：Codex/OpenAI skill 入口，也是最短通用方法说明。
- `AGENTS.md`：Codex、OpenAI Agents 和通用 Agent 可读入口。
- `CLAUDE.md`：Claude Code 入口。
- `GEMINI.md`：Gemini 入口。
- `.cursorrules`：Cursor 旧式规则入口。
- `.cursor/rules/ai-memory-palace.mdc`：Cursor 项目规则入口。
- `.clinerules`：Cline 入口。
- `.windsurfrules`：Windsurf 入口。

### 其他项目快速接入

在任意项目根目录执行：

```bash
<LOCAL_SKILL_WORKSPACE>/记忆宫殿SKILL/scripts/install-agent-entrypoints.sh /path/to/project
```

该脚本只写入短入口文件，指向本 SKILL 源目录，不复制完整资料。
