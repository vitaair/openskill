# 递交前核对清单

## 材料完整性

- [ ] 申请表已在线填报并导出终稿。
- [ ] 说明书或系统说明书已导出 PDF。
- [ ] 源程序代码鉴别材料已导出 PDF。
- [ ] 截图说明已导出 PDF。
- [ ] 主体证明扫描件已准备，企业申请须加盖公章。
- [ ] 委托代理时已准备真实代理委托书；自理登记不上传占位委托书。

## 一致性

- [ ] 软件全称在申请表、说明书、源程序页眉、PDF 文件名中一致。
- [ ] 版本号全套一致。
- [ ] 著作权人名称与营业执照一致。
- [ ] 开发完成日期、发表状态、首次发表日期与事实一致。
- [ ] 说明书功能模块、截图、源程序模块能互相对应。

## 格式

- [ ] PDF 为 A4 或符合登记系统当时要求。
- [ ] 页眉页脚和连续页码已检查。
- [ ] 源程序材料为可复制文本，不是截图。
- [ ] 源程序每页行数符合要求。
- [ ] 说明书图片清晰，图号和图注完整。

## 风险清理

- [ ] 删除“待补”“请按实际修改”“占位”等编辑提示。
- [ ] 删除测试账号、密码、密钥、生产连接串、手机号等敏感信息。
- [ ] 删除与本软件无关的父平台旧名、模板名或其他项目名。
- [ ] 不把内部说明、脚本说明、历史整改记录放入对外提交包。
