# Mobile Component Map

## NutUI Base Components

Use NutUI React for:

- Button
- Tabs
- Popup
- Toast
- Tag
- Badge
- Progress
- SearchBar
- Form
- Cell

## Business Components

Create project components:

- `MetricCard`: dashboard metric
- `TodoItem`: actionable todo
- `MessageItem`: notification row
- `ServiceCard`: service catalog item
- `StatusBadge`: business status
- `TemplatePreview`: notification template
- `FlowTimeline`: process guide
- `PermissionGuard`: RBAC display control
