# Blueprint Patterns

## Mobile Workbench

Use when target users need quick operational awareness.

Sections:

- Header status
- Metric grid
- Urgent todo list
- Quick actions
- Latest notifications

## Todo Center

Use for actionable items.

Common actions:

- View detail
- Complete
- Reject
- Confirm
- Upload material

## Message Center

Use for notifications.

States:

- unread
- read
- important
- archived

## Service Center

Use for service catalog and guidance.

Features:

- Category filter
- Service card
- Template preview
- Flow entry
